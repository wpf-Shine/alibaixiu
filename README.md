# 阿里百秀内容管理系统
分为内容管理和内容展示两大核心功能
## 功能模块
### 内容模块
|模块|功能|
|-|-|
|用户|登录、退出、用户增删改查|
|文章|文章管理|
|分类|分类管理|
|评论|评论管理|
|网站设置|关键字、描述、网站logo、轮播图|
### 内容展示
|模块|功能|
|-|-|
|首页|导航、文章数据展示|
|列表页|根据分类显示文章列表|
|详情页|文章详情数据展示、实现评论功能|

## 项目架构
